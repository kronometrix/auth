# Kronometrix Authentication #

This is the authentication binary package for **Kronometrix** software.

## Modules ##
* Openresty
* Redis

## Libs ##
* lua-resty-http-simple
* lua-resty-template
* router.lua
* inspect.lua (development mode)

### Required Libraries ###

FreeBSD
* openssl-1.0.1_13
* pcre-8.34_1

Linux
* libpcre3-dev
* libssl-dev
